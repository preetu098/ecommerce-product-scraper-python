# E-Commerce Product Scraper

A professional web scraping project built with Python, Requests, BeautifulSoup, and Pandas.

## Project Overview

This scraper extracts book information from the Books To Scrape website and stores the collected data in CSV format for further analysis.

## Features

- Extract Book Title
- Extract Price
- Extract Availability Status
- Extract Rating
- Extract Product URL
- Export Data to CSV

## Technologies Used

- Python 3
- Requests
- BeautifulSoup4
- Pandas
- lxml

## Installation

Clone the repository:

```bash
git clone https://github.com/preetu098/ecommerce-product-scraper-python.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Run the Project

```bash
python scraper.py
```
