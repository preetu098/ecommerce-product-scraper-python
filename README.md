# E-Commerce Product Scraper

A Python web scraping project built using Requests, BeautifulSoup, and Pandas.

## Features

- Extract book title
- Extract price
- Extract stock availability
- Export data to CSV

## Technologies

- Python
- Requests
- BeautifulSoup4
- Pandas

## Run

```bash
pip install -r requirements.txt
python scraper.py
```

## Output

CSV file generated in:

```text
data/books.csv
```
